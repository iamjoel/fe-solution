# vue3-admin

## 解藕
### 与外部解藕
通过加入中间适配层，而不是直接调用。
* 与第三方UI组件解藕。
* 与接口调用解藕。

### 与内部解藕
* 数据交 与 和UI 解耦。用 hooks。
* 配置与业务逻辑解藕。
