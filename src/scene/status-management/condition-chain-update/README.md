# 有条件的联动变化
## 宽随长变化的场景
做一个配置页面来记录纸板的长和宽。长和宽有如下约束：
1. 长，宽 的值可单独改。
2. 长的值改动后，宽的值会自动变成长的一半。
3. 可以设置当前的长宽值为默认值。
4. 可以将长宽值还原为默认值。此时，长的改动不会触发宽的值的改动。 

推荐 [声名式实现](./src/width-length.js) 而不是 [命令式实现](./src/width-length-command.js)。

## 宽随长，高随宽变化的场景
如果觉得这个还比较容易。可以再加入高这个变量。
1. 长，宽，高 的值可单独改。
2. 长的值改动后，宽的值会自动变成长的一半。
3. 宽的值变化后，高的值会自动变成宽的一半。
4. 可以设置当前的长宽高值为默认值。
5. 可以将长宽高值还原为默认值。
